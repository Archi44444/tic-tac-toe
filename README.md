# 🎮 Tic Tac Toe Game

A simple and fun Tic Tac Toe game built using HTML, CSS, and JavaScript. Play against a friend in the browser(desktop mode)!

## 🕹️ Features

- Two-player game
- Turn-based interaction
- Reset option
- GIF animation on winning (🎉)
